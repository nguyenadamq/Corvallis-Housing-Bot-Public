<h1>Welcome to the Duerksen Corvallis Housing Bot</h1>
The purpose of this program is to be able to pull from Duerksen's website with user defined criteria and optionally output to a user's discord API unfiltered and filtered webhook


<h2>User defined criteria:</h2>
<ul>
  <li>Minimum beds</li>
  <li>Minimum bathrooms</li>
  <li>Maximum rent price</li>
  <li>Minimum Square Feet</li>
  <li>Housing Type(Houses, Duplex, Condo) or (Include Apartments)</li>
</ul>

<h2>Current files needed to run:</h2>
<ul>
  <li>duerksen-bot.py</li>
</ul>
<h2>Run command to install dependencies:</h2>
pip install bs4 selenium requests discord_webhook fake_useragent
